# 概要
---
透過中介服務，您的應用程式就能放送眾多來源 (包括 AdMob 聯播網、第三方廣告聯播網和內部廣告活動) 的廣告。

# 完成串接指示
---
若您尚未完成串接廣告形式前的串接說明，請先前往[串接說明]完成相關設定，再透過 CocoaPods 將 Vpon AdMob Adapter 加入 App Project 中：

```
pod 'AdMobMediationAdapterVpadn'
```
